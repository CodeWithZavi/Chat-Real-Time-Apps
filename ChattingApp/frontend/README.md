# ChattingApp Frontend

React + Vite frontend for the real-time chat application.

## Environment

Create a `.env` file in this folder when running against anything other than local defaults:

```env
VITE_API_BASE_URL=http://localhost:5000/api
VITE_SOCKET_URL=http://localhost:5000
```

If these are not set, the app falls back to the local development endpoints above.
